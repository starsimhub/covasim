# Welcome to Covasim

Covasim is a stochastic agent-based simulator, written in Python, for exploring and analyzing the COVID-19 epidemic.

**Where should I start?**

- Take a quick look at the [Overview](overview.md) for a general introduction.
- When you're ready, the [Tutorials](tutorials/tut_intro.ipynb) will help you get started.
- For a specific feature or keyword, use search (top right).
- Questions? Email [info@covasim.org](mailto:info@covasim.org).
