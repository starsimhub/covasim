# Tutorials

These tutorials walk through how to use Covasim. To run locally, start a Jupyter environment in `docs/tutorials` (`jupyter lab` or `jupyter notebook`).

- [Getting started](tut_intro.ipynb) — Install, parameters, running, and plotting
- [Plotting](tut_plotting.ipynb) — Customizing figures and result views
- [Running simulations](tut_running.ipynb) — Batch runs, parallelism, and scenarios
- [People and populations](tut_people.ipynb) — Agents, networks, and population types
- [Interventions](tut_interventions.ipynb) — Testing, tracing, distancing, and vaccination
- [Analyzers](tut_analyzers.ipynb) — Recording and exporting custom outputs
- [Calibration](tut_calibration.ipynb) — Fitting parameters to data
- [Immunity](tut_immunity.ipynb) — Variants, vaccines, and waning immunity
- [Deployment](tut_deployment.ipynb) — Running at scale and in production
- [Tips and tricks](tut_tips.ipynb) — Shortcuts and common patterns
- [Advanced usage](tut_advanced.ipynb) — Lower-level APIs and extensions
